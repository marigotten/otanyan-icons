## Installation

To install the extension just execute the following command:

```sh
ext install otanyan-icons
```

## Usage
* `Linux` & `Windows` > **File > Preferences > File Icon Theme > Otanyan Icons**.
* `MacOS` > **Code > Preferences > File Icon Theme > Otanyan Icons**.
